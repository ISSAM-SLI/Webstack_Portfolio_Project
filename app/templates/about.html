About Quizify

Quizify is an interactive quiz platform built using Python and Flask. It offers a dynamic user experience where individuals can customize quizzes by selecting the number of questions, category, and difficulty level. The questions are fetched in real-time from the Open Trivia Database API, ensuring up-to-date and diverse content.

With features like form validation, progress tracking, and responsive design, Quizify delivers a seamless experience across all devices. After completing the quiz, users receive feedback to help improve their knowledge, including correct/incorrect answers and motivational insights.
